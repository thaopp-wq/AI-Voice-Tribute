# IIP032 v5.8.0 — Video Template with Audio Reference (Wedding Tribute Video)

**Version**: 5.8.0
**Feature ID**: IIP032
**Platform**: iOS (SwiftUI, iOS 26+)
**Status**: In Development
**Last updated**: 2026-05-18

---

## 1. Tổng quan

### Bối cảnh

User muốn tạo video tribute cho đám cưới — ví dụ người cha/mẹ đã mất nói lời chúc phúc tại đám cưới con. Hiện chưa có công cụ nào kết hợp đơn giản: ảnh khuôn mặt + giọng nói clone + lip-sync thành video ngắn cảm xúc.

### Giải pháp

Wizard tạo video theo template nằm trong tab **Moments**, segment **"Wedding"**. User chọn template, upload ảnh, cung cấp hoặc chọn giọng nói, tùy chỉnh script → hệ thống generate video lip-sync ~20 giây bằng **Seedance 2.0** (ByteDance qua fal.ai).

### Đầu ra

- Video lip-sync AI ~20 giây (1080p)
- Tạo từ: ảnh + giọng nói (upload hoặc AI voice tone) + script tùy chỉnh
- Script template đa ngôn ngữ (13 ngôn ngữ)

---

## 2. Luồng tổng quan

```
Moments Tab (Wedding segment)
    └── Duyệt template cards → Bấm "Get Video"
         └── Màn 1: Uploading Images
              └── Màn 2: Choosing Voice
                   └── Màn 3: Writing Message
                        └── Màn 4: Bringing Them Back (Processing)
                             └── Màn 5: A Moment Together (Result)
```

| # | Màn hình | Title trên app | Mục đích |
|---|----------|----------------|----------|
| 0 | Life Moments | Life Moments | Duyệt và chọn wedding template |
| 1 | Upload Photos | Uploading Images | Upload 2–3 ảnh (speaker, cô dâu/chú rể) |
| 2 | Upload Audio | Choosing Voice | Cung cấp giọng nói: upload file hoặc chọn AI voice sample |
| 3 | Edit Script | Writing Message | Tùy chỉnh script, nhập tên cô dâu/chú rể |
| 4 | Processing | Bringing Them Back | Hiển thị tiến trình tạo video (4 bước) |
| 5 | Output Preview | A Moment Together | Xem, chia sẻ, lưu, thích video đã tạo |

### 3.1 Uploading Images

**Nav title**: "Uploading Images"

**Giao diện**:
- Số slot ảnh theo template (`peopleNumber`: 2 hoặc 3)
- Mỗi slot có label từ template (ví dụ: "Upload a photo of the father")
- Slot trống: hình chữ nhật viền nét đứt (220x200) + icon plus + badge vàng "i"
- Slot đã có ảnh: preview ảnh (220x200, bo góc) + nút X xóa (góc trái trên) + warning vàng (nếu có)

#### Hành vi upload ảnh

**Given** một slot ảnh đang trống
**When** user bấm vào slot
**Then** mở PhotosPicker (thư viện ảnh hệ thống) để user chọn ảnh

**Given** user đã chọn ảnh từ PhotosPicker
**When** ảnh load thành công
**Then** hiển thị ảnh vào slot tương ứng và chạy kiểm tra chất lượng ảnh ở background

#### Hành vi kiểm tra ảnh (non-blocking)

**Given** ảnh đã được upload vào slot
**When** hệ thống phát hiện không có khuôn mặt nào (Vision + CIDetector)
**Then** hiển thị warning vàng: "No face detected. A clear face photo works best." (KHÔNG chặn, user vẫn tiếp tục được)

**Given** ảnh đã được upload vào slot
**When** ảnh có resolution thấp (width hoặc height < 200px)
**Then** hiển thị warning vàng: "Low resolution. A higher quality photo works best."

**Given** ảnh đã được upload và không có vấn đề
**When** kiểm tra hoàn tất
**Then** không hiển thị warning

#### Hành vi xóa ảnh

**Given** một slot đã có ảnh
**When** user bấm nút X (góc trái trên ảnh)
**Then** xóa ảnh khỏi slot, xóa warning tương ứng, slot trở về trạng thái trống

#### Hành vi nút Continue

**Given** chưa có ảnh nào được upload (tất cả slot trống)
**When** user nhìn nút "Continue"
**Then** nút ở trạng thái disabled (màu xám #2A2A2C, text #6B6B6E)

**Given** đã có ít nhất 1 ảnh được upload (bất kỳ slot nào)
**When** user nhìn nút "Continue"
**Then** nút ở trạng thái enabled (màu đỏ #8E0612, text trắng)

**Given** nút "Continue" đang enabled
**When** user bấm "Continue"
**Then** navigate đến màn Choosing Voice

### 3.2 Choosing Voice

**Nav title**: "Choosing Voice"

Flow quyết định trước (Decision-first UX): user phải trả lời có/không trước khi thấy các option.

#### Step 1: Câu hỏi quyết định (trạng thái ban đầu)

**Giao diện**: Card giữa màn hình chứa:
- Câu hỏi hero: *"Do you have a recording of Dad's/Mom's voice?"*
- Subtitle: *"A real voice will make the tribute feel truly personal."*
- Waveform animation (3 lớp sine wave)
- 2 nút CTA
- Nút "Continue" ẩn ở step này

**Given** user vào màn Choosing Voice lần đầu
**When** màn hình hiển thị
**Then** hiển thị card câu hỏi, nút "Continue" bị ẩn

**Given** user đang ở step câu hỏi
**When** user bấm "No, create one for me"
**Then** chuyển sang step Voice Sample, tự động chọn voice tone đầu tiên

**Given** user đang ở step câu hỏi
**When** user bấm "Yes, I have a recording"
**Then** chuyển sang step Upload

#### Step 2A: Upload giọng nói (hasRealVoice = true)

**Giao diện**:
- Title: *"Upload Dad's/Mom's voice"*
- Option 1: "Upload audio" — icon mic + subtitle "WAV or MP3 from your device"
- Option 2: "Use a voice from video" — icon film + subtitle "We'll extract the audio track"

**Given** user đang ở step Upload
**When** user bấm "Upload audio"
**Then** mở document picker (hỗ trợ WAV, MP3, M4A)

**Given** user chọn file audio từ document picker
**When** file có định dạng hợp lệ (WAV/MP3/M4A)
**Then** copy file vào thư mục tạm, hiển thị audio loaded state

**Given** user chọn file audio
**When** file có định dạng không hợp lệ
**Then** hiển thị lỗi: "Unsupported format. Please upload a WAV or MP3 file."

**Given** user đang ở step Upload
**When** user bấm "Use a voice from video"
**Then** mở PhotosPicker ở chế độ chọn video

**Given** user chọn video từ PhotosPicker
**When** video đang được xử lý
**Then** hiển thị indicator "Extracting audio..."

**Given** video đã được chọn
**When** extract audio thành công
**Then** hiển thị audio loaded state với file M4A đã extract

**Given** video đã được chọn
**When** video không có audio track
**Then** hiển thị lỗi: "No audio found in video. Please upload a different file."

#### Step 2B: Voice Sample (hasRealVoice = false)

**Giao diện**:
- Title: *"How do you remember Dad's/Mom's voice?"*
- Subtitle: *"Each voice carries a different warmth and emotion."*
- Danh sách 7 voice tone (radio-style, chỉ chọn 1)

**Given** user chuyển sang step Voice Sample
**When** màn hình hiển thị
**Then** hiển thị 7 voice tone, tone đầu tiên được chọn sẵn (viền đỏ)

**Given** danh sách voice tone đang hiển thị
**When** user bấm vào một tone
**Then** tone đó được chọn (viền đỏ), các tone khác bỏ chọn

**Given** một voice tone đang hiển thị
**When** user bấm nút play (hình tam giác) bên phải tone
**Then** phát demo giọng nói TTS với pitch/rate phù hợp tone đó

**Given** đang phát preview một tone
**When** user bấm nút play tone khác
**Then** dừng preview cũ, phát preview mới

#### Audio Loaded State

**Given** audio file đã được load thành công
**When** màn hình hiển thị
**Then** hiển thị card audio: tên file, định dạng, thời lượng, waveform, nút play/pause, link "Replace"

> **Lưu ý**: Audio quality validation (noise, multi-voice, dynamic range, thời lượng) đã chuyển sang server-side. Mobile chỉ check format file (WAV/MP3/M4A). Seedance 2.0 service sẽ validate và reject nếu audio không đạt → mobile hiển thị lỗi từ server response.

#### Nút Back

**Given** đang ở audio loaded state
**When** user bấm nút back
**Then** xóa audio, quay về step Upload

**Given** đang ở step Upload hoặc Voice Sample
**When** user bấm nút back
**Then** quay về step câu hỏi quyết định

**Given** đang ở step câu hỏi quyết định
**When** user bấm nút back
**Then** dismiss màn hình (quay về Uploading Images)

#### Nút Continue

**Given** user đã có audio source hợp lệ (file không lỗi validation HOẶC đã chọn voice tone)
**When** user bấm "Continue"
**Then** navigate đến màn Writing Message

### 3.3 Writing Message

**Nav title**: "Writing Message"

**Giao diện** (các section ngăn cách bằng divider mỏng):

1. **Hero**: Title "A message for wedding day" + subtitle "Create a loving message in their voice, for the moment they couldn't be there."
2. **Names**: Title "Who is getting married?" + 2 field ngang nhau "Bride's name" / "Groom's name"
3. **Script**: Title "A message from Dad/Mom" + text editor + word count + link "Restore original wording"
4. **Footer note**: Ghi chú về độ dài script

#### Hành vi nhập tên

**Given** script chứa placeholder `{bride_name}`
**When** user nhập tên cô dâu (ví dụ "Emma")
**Then** tất cả `{bride_name}` trong script tự động thay bằng "Emma"

**Given** script đã chứa tên cô dâu "Emma"
**When** user sửa tên thành "Anna"
**Then** tất cả "Emma" trong script tự động thay bằng "Anna"

**Given** script đã chứa tên cô dâu "Emma"
**When** user xóa hết tên (field trống)
**Then** tất cả "Emma" trong script tự động thay lại bằng `{bride_name}`

(Tương tự cho `{groom_name}` / tên chú rể)

#### Hành vi script editor

**Given** script đang ở chế độ hiển thị (display mode)
**When** user bấm vào vùng script
**Then** chuyển sang chế độ chỉnh sửa (edit mode), bàn phím xuất hiện

**Given** script có nhiều hơn 50 từ
**When** đang ở chế độ hiển thị
**Then** các từ từ vị trí 51 trở đi được tô vàng (#E8C840) với background vàng nhạt

**Given** script đang được chỉnh sửa
**When** user bấm ra ngoài vùng script
**Then** chuyển về chế độ hiển thị, ẩn bàn phím

#### Hành vi word count

**Given** script có <= 50 từ
**When** hiển thị word count
**Then** hiển thị "XX/50 words" màu xám + footer "Shorter messages usually sound more natural and emotional."

**Given** script có > 50 từ
**When** hiển thị word count
**Then** hiển thị "XX/50 words" màu đỏ (#F26B8C) + footer "Your message is too long. Please keep it under 50 words for best results." (màu đỏ)

#### Hành vi Restore

**Given** user đã chỉnh sửa script
**When** user bấm link "Restore original wording"
**Then** script trở về nội dung mặc định của template (đã thay tên nếu có nhập)

#### Hành vi đa ngôn ngữ

**Given** user đặt ngôn ngữ thiết bị là tiếng Việt
**When** chọn template
**Then** script mặc định hiển thị bằng tiếng Việt

**Given** ngôn ngữ thiết bị không nằm trong 13 ngôn ngữ hỗ trợ
**When** chọn template
**Then** script mặc định hiển thị bằng tiếng Anh (fallback)

Ngôn ngữ hỗ trợ: en, vi, ja, ko, zh, fr, hi, es, pt, ru, id, fil, bn

#### Nút "Create video"

**Given** script không trống (sau khi trim whitespace)
**When** user bấm "Create video"
**Then** navigate đến màn Processing

**Given** script trống
**When** user nhìn nút "Create video"
**Then** nút ở trạng thái disabled

### 3.4 Bringing Them Back (Processing)

**Nav title**: "Bringing Them Back"

**Giao diện**:
- Avatar tròn (130px) với thumbnail template
- Vòng tròn progress (160px) bao quanh avatar — fill theo chiều kim đồng hồ từ 12 giờ
- Vòng pulse ngoài cùng (176px) với animation nhấp nháy
- Text hero: *"We're bringing them back to you"* (Cormorant Bold 28pt)
- Danh sách 4 step (xuất hiện tuần tự)

| Step | Label (đang xử lý) | Label (hoàn thành) |
|------|---------------------|---------------------|
| 1 | Remembering their face... | Remembering their face |
| 2 | Listening to their voice... | Listening to their voice |
| 3 | Bringing their words to life... | Bringing their words to life |
| 4 | They're almost here... | They're almost here |

#### Hành vi processing

**Given** user vào màn Processing
**When** chưa có processing nào đang chạy
**Then** tự động bắt đầu processing, step 1 chuyển sang trạng thái "in progress"

**Given** processing đang chạy
**When** progress đạt 25%
**Then** step 1 → completed (text mờ + checkmark), step 2 → in progress (text trắng + spinner)

**Given** processing đang chạy
**When** progress đạt 50%
**Then** step 2 → completed, step 3 → in progress

**Given** processing đang chạy
**When** progress đạt 75%
**Then** step 3 → completed, step 4 → in progress

**Given** processing đang chạy
**When** progress đạt 100%
**Then** tất cả step → completed, tự động navigate đến màn Result

#### Hành vi lỗi

**Given** processing đang chạy
**When** xảy ra lỗi
**Then** hiển thị error modal overlay với title "Generation Failed", message "Something went wrong while generating your video. Please try again.", nút "Back to Home"

**Given** error modal đang hiển thị
**When** user bấm "Back to Home"
**Then** hủy processing, pop về màn Life Moments

#### Trạng thái step

| Trạng thái | Hiển thị |
|-------------|----------|
| Pending | Ẩn (opacity 0) |
| In progress | Text trắng + spinner |
| Completed | Text mờ (50% opacity) + checkmark |
| Failed | Text hồng (#F26B8C) + icon X |

### 3.5 A Moment Together (Result)

**Nav title**: "A Moment Together"

**Giao diện**:
- Nav bar trailing: nút heart (trái) + nút download (phải)
- Video player full-bleed (tỉ lệ 9:16)
- Nút play/pause glass (giữa video)
- Thanh playback controls: nút flag | link "Full screen" | nút mute/unmute
- Nút action: "Share" (capsule glass) + Recreate (circle glass, icon arrow.clockwise)

#### Hành vi heart/like

**Given** video đang hiển thị, heart chưa được bấm
**When** user bấm nút heart trên nav bar
**Then** icon chuyển từ `heart` (trắng) sang `heart.fill` (hồng #FF4D6A) với spring animation + scale 1.15x

**Given** heart đang ở trạng thái liked (filled)
**When** user bấm nút heart lần nữa
**Then** icon chuyển về `heart` (trắng), scale về 1.0

#### Hành vi download/save

**Given** user bấm nút download (arrow.down.to.line)
**When** chưa có quyền truy cập Photos
**Then** hiển thị popup xin quyền Photos library

**Given** đã có quyền Photos
**When** lưu thành công
**Then** hiển thị toast "Saved to Photos" (1.8 giây rồi tự ẩn)

**Given** đã có quyền Photos
**When** lưu thất bại
**Then** hiển thị toast "Save failed"

**Given** user từ chối quyền Photos
**When** xử lý response
**Then** hiển thị toast "Photos access denied"

#### Hành vi Share

**Given** user bấm nút "Share"
**When** xử lý action
**Then** mở system share sheet (UIActivityViewController) với nội dung video + text "Made with Artimind"

#### Hành vi Recreate

**Given** user bấm nút Recreate (arrow.clockwise)
**When** xử lý action
**Then** hủy processing, pop toàn bộ flow về màn Life Moments

#### Hành vi play/pause

**Given** video đang dừng
**When** user bấm nút play (giữa video)
**Then** icon chuyển sang pause, video phát

**Given** video đang phát
**When** user bấm nút pause
**Then** icon chuyển sang play, video dừng

#### Hành vi mute/unmute

**Given** video đang có âm thanh
**When** user bấm nút speaker
**Then** icon chuyển sang `speaker.slash.fill`, tắt tiếng video

**Given** video đang tắt tiếng
**When** user bấm nút speaker
**Then** icon chuyển sang `speaker.wave.2.fill`, bật tiếng video

## 4. Data Model

### WeddingTemplate

| Field | Type | Mô tả |
|-------|------|-------|
| `id` | String | ID duy nhất (ví dụ: "wedding_father_speech_01") |

| `thumbnail` | String | Tên ảnh asset |
| `videoPreview` | String | URL video preview (tương lai) |
| `audioPreview` | String | URL audio preview (tương lai) |
| `speakerRole` | SpeakerRole | .father / .mother |
| `peopleNumber` | Int | Số slot ảnh (2 hoặc 3) |
| `scenePrompt` | String | Mô tả cảnh cho AI |
| `backgroundPrompt` | String | Mô tả background cho AI |
| `nonSpeakerPrompt` | String | Mô tả hành vi người không nói |
| `scriptTemplates` | [String: String] | Mã ngôn ngữ → nội dung script |

| `photoLabels` | [String] | Label cho từng slot ảnh |
| `speakerPhotoSlot` | Int | Slot nào là speaker |
